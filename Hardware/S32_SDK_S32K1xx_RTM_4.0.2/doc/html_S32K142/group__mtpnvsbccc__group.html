SBC configuration control options (0x74)
System Basis Chip library (SBC) - UJA113x Family » 07. MTPNV and ID

Detailed Description

This section lists all the possible options of 0x74 register related to:

  1. V1 reset threshold at start-up configuration
  2. FNMC mode configuration
  3. SDMC mode configuration
  4. VEXT pin mode selection
  5. Sleep mode support configuraion
See also
