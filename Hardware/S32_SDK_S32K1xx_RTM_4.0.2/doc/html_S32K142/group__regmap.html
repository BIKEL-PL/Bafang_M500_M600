Register address map
System Basis Chip library (SBC) - UJA113x Family » 00. Library internals

Detailed Description

The SPI protocol requires the first byte transmitted on MOSI line to be set to register address the user wants to access (write/read). The byte consists of ADDRESS[7:1] and READ_BIT[0]. This section lists all the register addresses available on UJA113x devices which can be used throughout the source code instead of fixed values. The defined addresses are already shifted.
The address is useful in UJA113X_SbcTransfer() function, which allows generic read/write access.
