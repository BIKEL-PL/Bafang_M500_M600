Start-up control configuration options (0x73)
System Basis Chip library (SBC) - UJA113x Family » 07. MTPNV and ID

Detailed Description

This section lists all the possible options of 0x73 register related to:

  • Reset pulse width on RSTN pin
  • V2 start-up configuration
  • HVIO 4 - 2 pin mode selection
