Bank N fail interrupt enable configuration options (0x38/0x48)
System Basis Chip library (SBC) - UJA113x Family » 06. Interrupts

Detailed Description

This section lists all the possible options of 0x38/48 registers related to:

  • HVIO-N short-circuit interrupt
  • HVIO-N open load interrupt
Note
As HVIO bank 0 & 1 has the same memory layout, these macros can be used for Bank 0 & Bank 1 fail interrupt configuration
See also
UJA113X_SetBankFailInterrupts()
