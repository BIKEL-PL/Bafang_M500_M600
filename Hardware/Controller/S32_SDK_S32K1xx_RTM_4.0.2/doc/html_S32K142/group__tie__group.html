Transceiver interrupt enable configuration options (0x23)
System Basis Chip library (SBC) - UJA113x Family » 06. Interrupts

Detailed Description

This section lists all the possible options of 0x23 register related to:

  • CAN bus silence interrupt
  • LIN 2 wake-up interrupt
  • LIN 1 wake-up interrupt
  • CAN failure interrupt
  • CAN wake-up interrupt
