Supply interrupt enable configuration options (0x1C)
System Basis Chip library (SBC) - UJA113x Family » 06. Interrupts

Detailed Description

This section lists all the possible options of 0x1C register related to:

  • SMPS status interrupt
  • Battery monitor overvoltage interrupt
  • Battery monitor undervoltage interrupt
  • VEXT overvoltage interrupt
  • VEXT undervoltage interrupt
  • V1 undervoltage interrupt
