Bank N wake-up interrupt enable configuration options (0x37/0x47)
System Basis Chip library (SBC) - UJA113x Family » 06. Interrupts

Detailed Description

This section lists all the possible options of 0x37/47 registers related to:

  • HVIO-N falling edge interrupt
  • HVIO-N rising edge interrupt
Note
As HVIO bank 0 & 1 has the same memory layout, these macros can be used for Bank 0 & Bank 1 wake-up interrupt configuration
See also
UJA113X_SetBankWupInterrupts()
